# Ohana Landing Page Reference

Reference document for Nivasesa landing page design based on [Ohana](https://liveohana.ai/) analysis.

## Company Overview

**Ohana** (liveohana.ai) - Student housing sublet platform
- Founded: 2023
- Raised: $4.15M
- Advisors: Spencer Rascoff (Zillow co-founder), Surabhi Gupta (Airbnb Director of Engineering)
- Coverage: 6 major cities, 10 universities
- Vision: "Make the chaos of subleasing as easy as booking an Airbnb"

## Target Audience

- College students seeking sublets
- Interns needing temporary housing (30 days - 12 months)
- International students (emphasize beginner-friendly UX)

---

## Landing Page Structure

### 1. Hero Section
**Key Elements:**
- Clear value proposition headline
- Subheadline explaining the service
- Search bar with location input
- CTA buttons: "Find a Room" / "List Your Room"
- Background: Property imagery or illustrations

**Ohana Approach:**
- "Sublets, Short Term Rentals, and Rooms for Rent"
- Emphasis on ease and safety
- City selector prominent

**Nivasesa Adaptation:**
- "Find Your Perfect Room with People Like You"
- Freedom Score highlight
- Cultural compatibility emphasis
- Cities: Dallas, Austin, Houston, San Jose, Jersey City

### 2. How It Works Section
**3-Step Process (for Seekers):**
1. **Browse Listings** - Filter by location, price, move-in date, roommate info
2. **Connect & Chat** - Message hosts, schedule video calls
3. **Book with Confidence** - Verified listings, secure deposits

**3-Step Process (for Hosts):**
1. **Create Your Listing** - Add photos, describe your space
2. **Meet Potential Roommates** - Review applications, chat with candidates
3. **Find Your Match** - Accept applications, handle logistics

### 3. Trust & Safety Section
**Key Trust Signals:**
- ID verification badges
- Platform-supervised interactions
- Secure deposit handling
- "No more scams, questionable leases, or stolen security deposits"
- Professional team reviews all listings

**Nivasesa Adaptation:**
- Community trust model
- Cultural compatibility verification
- Realtor network backing
- Freedom Score transparency

### 4. Features Showcase
**Ohana Features:**
| Feature | Description |
|---------|-------------|
| Smart Filters | Location, neighborhood, move-in date, rooms, roommate info |
| Roommate Profiles | Current roommates displayed with intros |
| Saved Searches | Preferences saved + instant notifications |
| 24/7 Forum Search | Tech searches subleasing forums automatically |
| Video Calls | Introductory calls between parties |
| Secure Deposits | Platform holds deposits in escrow |

**Nivasesa Unique Features:**
| Feature | Description |
|---------|-------------|
| Freedom Score | 0-100 lifestyle freedom rating |
| Cultural Matching | Language, diet, lifestyle compatibility |
| Realtor Network | Culturally-aligned real estate agents |
| Group Search | Coordinate housing with roommates first |

### 5. Social Proof Section
**Elements:**
- Trustpilot rating widget
- User testimonials with photos
- Statistics (listings, users, successful matches)
- Partner/advisor logos

**Testimonial Themes:**
- "Easy to use platform"
- "Super seamless process"
- "Beginner-friendly for international students"
- "User friendly with supervised interactions"
- "Found nice apartment, platform really user friendly"

### 6. City Coverage Section
**Ohana Cities:** NYC, SF/Bay Area, Seattle, LA, DC, Pittsburgh

**Nivasesa Cities:**
- Texas: Dallas, Frisco, Plano, Austin, Houston
- California: San Jose, Bay Area
- New Jersey: Jersey City, Edison

### 7. CTA Section (Bottom)
**Primary CTA:** "Start Your Search" or "List Your Room"
**Secondary CTA:** "Learn More" / "Contact Us"

### 8. Footer
- Navigation links (About, How It Works, FAQ, Contact)
- Legal links (Terms, Privacy, Safety)
- Social media icons
- Newsletter signup
- App store badges (if applicable)

## UI/UX Patterns

### Search & Filtering
- Location autocomplete
- Date picker for move-in
- Price range slider
- Room type checkboxes
- Roommate preferences filters

### Listing Cards
- Large hero image
- Price prominent
- Key details (bedrooms, move-in date)
- Roommate avatars
- Quick action buttons (Save, Message)

### Map Integration
- Interactive map with pins
- Cluster markers for zoom-out
- List/map toggle view
- Neighborhood boundaries

### Messaging
- In-app chat
- Real-time or near-real-time
- Read receipts
- File/image sharing

---

## Conversion Optimization

### Trust Builders
1. Verification badges on listings
2. "Protected by Ohana" messaging
3. Team review process highlight
4. User success stories

### Urgency/Scarcity
1. "X people viewing this listing"
2. "Listed 2 days ago"
3. "Only 1 room left"

### Reduce Friction
1. Social login (Google, etc.)
2. Save search without account
3. One-click message templates
4. Mobile-first design
